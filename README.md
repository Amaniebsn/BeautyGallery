# Interactive Image Gallery

## Overview
This project is a responsive React image gallery featuring interactive UI elements such as hover effects, search filtering, and dark mode toggle.

The goal was to practice component structure, state management, and user interface interactivity using React.

## Features
- Image zoom effect on hover
- Dynamic image filtering via search bar
- Dark mode / Light mode toggle
- Responsive layout (mobile & desktop)
- Clean component-based architecture

## Tech Stack
- React
- JavaScript (ES6+)
- HTML5
- CSS3
